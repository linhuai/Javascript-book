7.3 数组去重

方法一 使用 ES6 Set、Array.from

原理

  1. ES6 中新增了 Set 数据结构，类似于数组，但 它的成员都是唯一的 其构造函数接受一个数组作为参数

  2. ES6 中新增了 Array.from 静态方法，可以把类数组转为数组

代码

function unique (arr) {
  if (Object.prototype.toString.call(arr) !== '[object Array]') {
    return false
  }
  return Array.from(new Set(arr))
}
